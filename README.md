# Autonomous Coding Assistant

An **Autonomous Coding Assistant** built with **LangGraph**, designed to generate, analyze, and improve code efficiently. The assistant integrates **Streamlit UI** for an interactive user experience.

## 🚀 Features
- **Automated Code Generation**: Generates code based on user prompts.
- **Code Analysis & Debugging**: Identifies issues and suggests improvements.
- **Code Optimization**: Recommends best practices for efficiency.
- **Interactive UI**: Built with Streamlit for a seamless experience.

---

## 📦 Installation

### Prerequisites
Ensure you have the following installed:
- Python 3.8+
- pip
- Git

### Clone the Repository
```sh
git clone <your-repo-url>
cd <your-repo-name>
```

### Install Dependencies
```sh
pip install -r requirements.txt
```

---

## 🚀 Usage

### Run the Assistant
```sh
streamlit run app.py
```
This launches the **Streamlit UI**, allowing users to interact with the agent.

### Example Usage
```python
# Example: Generate a sorting function
prompt = "Write a Python function to sort a list using quicksort."
# The agent will generate and display the code
```
